# 🛰️ demo_tele

**demo_tele** is a front-end web dashboard designed to visualize **drone telemetry data**.  
It’s a demo project made to show **how the Ground Telemetry Station UI will look** in future drone system builds.

---

## 🚀 Live Demo

🔗 [madhav-sawant.github.io/demo_tele](https://madhav-sawant.github.io/demo_tele)

---

## 🧩 Overview

This project is a **front-end-only** demo built for visualization and concept design.  
It represents the interface of a **Ground Telemetry Station** — where real-time data from the drone will be displayed in future versions.

---

## 🛠️ Built With

- **HTML5** – Structure  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Dynamic content and UI behavior  

---

## 🖥️ Features

- Clean and responsive UI  
- Placeholder sections for:
  - GPS coordinates  
  - Drone status  
  - Sensor data (altitude, roll, pitch, etc.)  
- Simple design ready for backend integration  

---

## 📦 Future Updates

- Integration with **ESP32** for real-time telemetry  
- Backend for **data logging** and **waypoint control**  
- **Live map visualization** of flight data
